# Branch Prompt - 'BasicStart/Toby'

Here was the original prompt given to Google Studio:

```text
Create a basic web-app using express.
The file structure of the app should be split into 3 categories:
- Data - where data can be stored using .json formats
- Public - where index.html lives, along with any necessary css and js files.
- Server - where the server side code lives, such that the server can be started using npm start, along with any other commands that are necessary for a functional web app.
```

I then asked it to adjust the placeholder text to fit the theme of a net-worth tracker using the following prompt:

```text
The app should be for a net-worth application, adjust it so the data fits this theme.
```

I also noticed that the app was generated using TypeScript, so gave the following prompt to fit the specification:

```text
Use only JavaScript, not any derivatives such as TypeScript, React, etc.
```

These prompts provided me with a working final web-app that fit the brief.
