# PersonalNetWorth
Personal Net Worth and Debt tracker

## This branch

For this branch we used Claude AI's sonnet 4.6 model and used the following prompt to generate a basic framework for a networth tracking app:

```
create a basic web page with the following aspects. this will be used for the basis of a personal wealth tracking app further down the line.
- data folder, where data will be stored as json
- public folder to host index html, any css and js used
- server folder used for the server startup with npm start and any server side js needed for the api
```

When unzipped, the application template that sonnet generated ran out of the box. It provided us with an aesthetically pleasing dashboard. As requested, this page is static and serves as a mere visual base-line for features to be functionally implemented later on.
